# dataset

Same as A1 and A2. The dataset contains GitHub pull requests authored either by human developers or by AI coding agents, made available through the MSR 2026 Mining Challenge. 

Information about this dataset can be accessed from: 

- Mining Challenge - [link](https://2026.msrconf.org/track/msr-2026-mining-challenge?#Call-for-Mining-Challenge-Papers)

AIDev dataset preprint available:
- The Rise of AI Teammates in Software Engineering (SE) 3.0: How Autonomous Coding Agents Are Reshaping Software Engineering 
- [arxiv: paper](https://arxiv.org/abs/2507.15003)
- [view pdf](https://arxiv.org/pdf/2507.15003)
- [HuggingFace Dataset](https://huggingface.co/datasets/hao-li/AIDev)

How to Participate in the Challenge: First, familiarize yourself with the AIDev dataset:   
The details about the AIDev infrastructure and the data are provided in our [preprint](https://arxiv.org/abs/2507.15003)

- The dataset can be downloaded from either [Hugging Face](https://huggingface.co/datasets/hao-li/AIDev) or [Zenodo](https://zenodo.org/records/16919272).

- GitHub [example code & notebooks](https://github.com/SAILResearch/AI_Teammates_in_SE3)

- An example Jupyter notebook demonstrating how to load and analyze the dataset is available [here](https://github.com/SAILResearch/AI_Teammates_in_SE3/blob/main/analysis/load_AIDev.ipynb), you can also open it directly in [Google Colab](https://colab.research.google.com/github/SAILResearch/AI_Teammates_in_SE3/blob/main/analysis/load_AIDev.ipynb).
